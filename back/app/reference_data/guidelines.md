# Referencias para revisión de OIT

Este directorio contiene información de referencia para la IA.

## Criterios mínimos
- Identificador del OIT presente.
- Fecha del documento y autor responsable.
- Alcance y propósito claramente definidos.
- Lista de requisitos y evidencias de cumplimiento.
- Firmas o validación del responsable.

## Reglas de validación
- El documento debe estar completo y legible.
- Debe existir coherencia entre requisitos y evidencias.
- El formato puede ser PDF o texto.
- Si falta un criterio mínimo: marcar `error`.
- Si hay dudas o riesgos: marcar `alerta`.
- Si todo está correcto: marcar `check`.
